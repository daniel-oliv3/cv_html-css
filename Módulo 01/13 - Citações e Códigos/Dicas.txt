tags
- code

- pre

- q 
 tag para sitação("sitações") " "

 tag para sitações em bloco (sitações muito longa como links etc..) para direitos autorais, citações de textos e livros

 tag para Abreviações, quando passa o curso do mouse no texto aparace a texto completo. (mostra a sigla completa)

- bdo
 tag para inverter o texto de tras para frente(utilizando o parametro dir"") o rtl mostra invertido, o ltr mostra normalmente
